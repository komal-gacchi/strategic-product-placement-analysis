# Strategic Product Placement Analysis

This project analyzes the impact of product placement on sales using Tableau visualization.

## Tools Used
- Tableau
- Python
- Excel Dataset

## Features
- Sales trend analysis
- Product category insights
- Regional sales visualization

## Dataset
Kaggle Superstore Dataset
